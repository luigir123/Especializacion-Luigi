# Repositorio de Especialización en Big Data

Este repositorio está destinado a almacenar y organizar datos, materiales, ejercicios y recursos relacionados con la especialización en Big Data. Aquí podrás guardar datasets, scripts, notebooks, documentación y cualquier otro archivo relevante para el desarrollo de proyectos y actividades de la especialización.

## Objetivo
Facilitar el acceso y la gestión de la información utilizada durante la especialización, promoviendo el orden y la colaboración entre los participantes.
